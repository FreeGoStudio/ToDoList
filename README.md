# ToDoList (Avalonia)

跨平台的 ToDoList 桌面应用（基于 Avalonia / C#）。

## 目标
提供一个简洁、可扩展、社区驱动的任务管理应用，支持 Windows / macOS / Linux。

## 快速开始（开发者）
前提：安装 .NET SDK 9.0 或更高（参见 https://dotnet.microsoft.com/）

1. 克隆仓库
   git clone https://github.com/FreeGoStudio/ToDoList.git
2. 进入项目目录
   cd ToDoList
3. 恢复依赖并构建
   dotnet restore
   dotnet build
4. 运行（示例）
   dotnet run --project src/ToDoList.App/ToDoList.App.csproj

## 运行测试
dotnet test
